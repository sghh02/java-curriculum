## 練習問題1

commons-langライブラリのJAR ファイル「commons-langjar」が、
c:*javalibフォルダにあるとします。
このライブラリを利用しているソースコードMain.javaをコンパイルして実行するために入力するコマンドを、「」を埋めて完成させてください(なお、Main.java 以外にはソースファイルは作成していないものとする)。

**コンパイル**

```java
> javac 「 1 」 ️Main.java
```

**実行**

```java
> java 「 2 」　Main
```
