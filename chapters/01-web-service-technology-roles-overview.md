## 全体像（超ざっくり）

あなたが作るのは、主に「バックエンド（Spring Boot）」です。

```text
ブラウザ / スマホアプリ
  ↓ HTTPリクエスト（URL, メソッド, JSON など）
Spring Boot（Java）
  ↓ Service / Repository
DB（データ保存）
  ↑
HTTPレスポンス（ステータスコード, JSON など）
```

## 最低限おさえる用語（暗記不要）

- **リクエスト / レスポンス**：送る / 返す
- **HTTP**：通信のルール
- **URL / パス**：どの機能にアクセスするか
- **ステータスコード**：結果の番号（例：200/400/404/500）
- **IPアドレス / ドメイン**：住所（数字）/ 名前（人間向け）
- **DNS**：ドメイン → IP の変換
- **フロントエンド**：画面側（HTML/CSS/JavaScript）
- **バックエンド**：ロジック＋データ側（Java/Spring Boot）
- **データベース**：データを保存する仕組み
