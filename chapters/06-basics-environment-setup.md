# 環境構築


ここからは実際にJavaのプログラムを動かしながら、プログラムの書き方を勉強していきます。
まず、カリキュラムを行うのに必要な環境構築をしてもらいます。
これは、実際の開発現場でも最初に行う作業です。
Javaを動かす環境と、課題を提出できる環境を作成してみましょう。

## 環境構築とは

・環境を作る作業のこと
・「環境」が何かはケース・バイ・ケースだよ
・実際にやる作業はケース・バイ・ケースだよ

今回行う環境構築は以下になります。
・Javaでプログラミングを行うための環境構築

## Javaでプログラミングを行うための環境構築

## IDE のインストール

**結合開発環境**とは

・開発環境だよ
・開発する上で必要なものが(ほぼ)全部詰め込まれているよ
・「IDE」と表現された場合は「Integrated Development Environment」の略だよ
・このカリキュラムでは**Eclipse**を使用します。

## Eclipseのインストール

**Eclipse**の特徴
・無料
・実際の開発現場で多く使われている
・開発の効率化
・標準でJavaの開発環境が含まれている
・プラグインを導入することでさまざまな言語が使える
・便利なプラグインが豊富
・開発支援機能
・コード編集機能

## インストール手順

[Java 統合開発環境 Eclipse 日本語化プロジェクト - Pleiades](https://willbrains.jp/)

上記ページからインストールします。

## Windows へのインストール


1.「最新版(画像では2023)」をクリックする。


2.「最新版(画像では2023)」をクリックすると、上記の画面が表示されますので、縦が「Java」、横が「Windows x64」=>「Full Edition」の「Download」ボタンをクリックしてください。


3.ダウンロードしたexeファイルをダブルクリックすると、上記の画面が表示されるので、上長などから作成先の指定がなければ、そのまま「解凍」ボタンを押下します。


4.解凍した「pleiades」フォルダの中の「20xx-xx」フォルダにある「eclipse」フォルダを選択すると、「eclipse.exe」がありますので、このファイルをダブルクリックします。
「インストールしようとしているアプリは、Microsoft 検証済みアプリではありません 」画面が出てきたら以下ページの手順で「eclipse.exe」を実行する。

[03_Windows11でMicrosoft Store以外からアプリをインストールする方法](https://www.notion.so/03_Windows11-Microsoft-Store-52cda956b1e3476e86bc0d18dcf79b66?pvs=21)


5.起動後「Eclipse IDE ランチャー」ダイアログが表示されますので、ワークスペースのフォルダを選択してください。
ワークスペースとは、アプリケーションのソースファイルや設定ファイルなどを保存するフォルダです。
デフォルトで表示されているフォルダを変更しても問題ありません。
ワークスペースを選択したら、そのまま「起動」ボタンを押下します。

これでEclipseが起動します。
起動すると、Eclipseの開発画面が表示されます。
これでEclipseが起動します。
起動すると、Eclipseの開発画面が表示されます。

Eclipseはタスクバーにピン留めしておきましょう！
やり方は、タスクバーのEclipseアイコンを右クリックで「タスクバーにピン留めする」です。

## macOS へのインストール


1.「最新版(画像では2023)」をクリックする。


2.「最新版(画像では2023)」をクリックすると、上記の画面が表示されますので、縦が「Java」、横が「Mac」=>「Full Edition」の「Download」ボタンをクリックしてください。


3.ダウンロードしたdmgファイルをダブルクリックし、「Eclipse」アイコンを「Applications」にドラッグアンドドロップしてください。


4.Eclipseの起動確認をしておきましょう。
「Applications」の「Eclipse」アイコンをダブルクリックします。


5.初回起動時、上記の警告が表示された場合、「OK」をクリックします。


6.警告が表示された場合、「システム環境設定」=>「プライバシーとセキュリティ」を聞きます。


7.「App Storeと確認済みの開発元からのアプリを許可」を選択し、「このまま開く」をクリックします。


8.許可するためにTouch ID か パスワードを入力してください。
次回以降は、普通に「Eclipse」アイコンをダブルクリックすることで、警告が表示されず、Eclipseが起動するようになります。


9.「Eclipse IDE ランチャー」ダイアログが表示されますので、ワークスペースのフォルダを選択してください。
ワークスペースとは、アプリケーションのソースファイルや設定ファイルなどを保存するフォルダです。
デフォルトで表示されているフォルダを変更しても問題ありません。
ワークスペースを選択したら、そのまま「起動」ボタンを押下します。

これでEclipseが起動します。
起動すると、Eclipseの開発画面が表示されます。
これで、Eclipseの起動確認は完了です。

## 予測変換機能をいれる

この機能はコードを書いているとと、途中で予測変換してくれるという便利機能です。

### 導入方法

Eclipseを開き以下手順を行ってください
環境設定(設定)→Java→エディター→コンテンツアシスト

「自動有効化を使用可能にする」にチェックを入れて自動有効化遅延を80にします。
Javaの有効化トリガーを「.abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ_」にします。

以上で完了です。
