## 練習問題1

コマンドライン引数にファイル名を与えると、 その拡張子を取り出すプログラムを作成してください。
 拡張子とは、ファイル名が「cmd.exe」であれば「exe」である。 java.sun.exe のように「.」が2つ以上ある場合もあるので、 拡張子は、最後の「.」の直後から末尾までの文字列、 と定義することができる。
main メソッドのあるクラスのクラス名は ExtensionExtractor とする。
