# Javaでできること


## Javaを使ってできること

**Java** とはプログラムを作るために利用するプログラミング言語の1つです。
Javaを使えば、さまざまなコンピュータで動作する多様なプログラムを開発する事ができます。
・Androidアプリケーションの開発
・ウィンドウアプリケーションの開発
・Webアプリケーションの開発
・コマンドラインアプリケーションの開発

次のような特徴から、Javaはさまざまな分野で利用されています。
・学びやすく標準的な基本文法
・大規模開発を支援するオブジェクト指向に対応
・豊富に準備された便利な命令群
・多様なコンピュータで同じように動作する汎用性

このような特徴があるため初めてプログラミング言語に触れる際にぴったりな言語となっています。
Javaを実際に手を動かしてプログラミングすることで、その強力さや汎用性を体感できるでしょう。
是非、Javaをマスターしていきましょう！
